Live CPU usage

This project shows your live CPU usage in 1 second intervals represented on a line graph. This was a fun project to make because I got to work with with some really cool technologies like Reacharts.js, OS utils, and WebSockets!!

Workout.Map()

Track your workouts on a map of your actual location. This project utilizes a free api by Leaflet as well as built in geolocation on the client's browser to achieve an aesthetic progress tracker.

Bankist App

This is a really fun dashboard that simulates an online bank account. This project is able to view current account balance, transaction history, as well as what was earned and what was spent. There is even a function to transfer funds between accounts.
